# Deskripsi Aplikasi

**Sistem Penjadwalan Prioritas Karyawan** adalah aplikasi berbasis **Node.js + Express + Sequelize (SQLite)** yang bertujuan untuk membantu tim HRD menentukan prioritas penjadwalan shift atau insentif karyawan berdasarkan kriteria kinerja utama, yaitu **Jabatan, Performa, Lama Kerja, dan Fleksibilitas**.

Aplikasi ini menerapkan konsep:
* **Object-Oriented Programming (OOP)**
* **Relational Database** menggunakan Sequelize ORM
* **Web Application** dengan tampilan dinamis (EJS)

Aplikasi dibuat sebagai **Final Project / Project UAS Mata Kuliah Logika Pemrograman**.

## Teknologi yang Digunakan

* **Node.js**
* **Express.js**
* **Sequelize ORM**
* **SQLite** (Database)
* **EJS** (Embedded JavaScript Templates)
* **Bootstrap 5** (CSS Framework)
* **Git & GitHub**
